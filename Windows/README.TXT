TLClipeditは、簡単なテキスト形式のクリップボードの内容を編集、閲覧する作業を簡単にするツールです。
以下の機能をサポートしています。
・クリップボードのテキスト内容の表示
・クリップボードにテキストを書き込み
・一時的に履歴として、クリップボードの内容を留める


テスト済みOS
Windows 10

インストール手順
アーカイブを適当な場所に解凍し、解凍したファイル内の、「prepare.exe」を実行してください。
prepare.exeは、必ずこのprepare.exeのあるディレクトリで実行してください。例えば、「C:\Users\User\Desktop\tlclipeditforwin」ディレクトリにprepare.exeがある場合は、「C:\Users\User\Desktop\tlclipeditforwin」ディレクトリに移動してから実行してください。間違っても、例えば、「C:\」上で、「C:\Users\User\Desktop\tlclipeditforwin\prepare.exe」なんてことはしないでください。
もしこの文の意味がわからなかったら、ただ単にエクスプローラ上でprepare.exeをダブルクリックしてください。おそらく、場合によっては、ただ単に「prepare」とだけ表示されていることもあります。
「prepare.exe」の実行後、「run_tlclipedit.bat」を実行すると、ソフトが起動します。

同梱のgtk+-bundle_3.6.4について
このソフトウェアは、ランタイムとしてgtk+-bundle_3.6.4を利用しています。ライセンスは、解凍後のディレクトリの、「share\doc\」下にそれぞれの付属したパッケージのドキュメントがあります。それらをご覧ください。
なお、オリジナルのファイルに関しては、以下から入手可能です。
http://www.tarnyko.net/dl/gtk.htm
